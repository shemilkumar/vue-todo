## Technologies

* Vue JS
* Pinia (State Management)
* Tailwind CSS
* Typescript
* Axios
* Vite (development tool)
* Git

## Features Included
* Can Add new Todo
* Drag and Drop todos
* Move to In Progress list
* Move to Completed list
* Remove todos
* Track progress
* Deadline indicator
* Dark mode

## Install Dependencies and Run

```
  npm install
  npm start
```
